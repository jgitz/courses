- you should git ignore the node_modules folder because of enormous size of the folder

  - goto project folder

    - create a new file named .gitignore

    - open gitignore folder

      - add the following listner

        node_modules/

        ** now the git ignores the node_modules folder

        **** / is important to denote folder 



- you should never copy the node modules file to recreate the project else where

- to recreate the project in another device... 

  - copy project folder except node_modules folder

  - paste it inside new device

  - open terminal from project folder and type

    :: npm i

      ** now npm downloads all dependencies from the package.json file
