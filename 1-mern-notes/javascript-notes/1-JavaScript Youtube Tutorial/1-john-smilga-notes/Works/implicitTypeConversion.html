In js we can make numbers as string by enclosing number values in quotes. If there are 2 variables with numbers as string and we do subtraction, multiplication, division or modulus, we get normal arithematic result. This is because of implicit type conversion in js. i.e the string numbers gets converted to number datatype automatically. BUT while doing ADDITION we WON'T get arithematic result. Instead 2 string numbers concatenates to create a string.

In subtraction, multiplication, division and modulus, the result gets converted to NUMBER datatype even though the variables are string numbers

In case of addition, the result gets converted to STRING datatype.
